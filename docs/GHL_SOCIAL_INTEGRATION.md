# GHL Social Media Integration

## Overview

The Publishare CMS now integrates with GoHighLevel (GHL) for automated social media posting. This allows articles generated by `agentic-content-gen` to be automatically posted to social platforms via GHL.

## Architecture

```
agentic-content-gen
    ↓ (generates article + social posts)
    ↓ (if share_to_social: true)
ghl-social-poster
    ↓ (fetches GHL config from database)
    ↓ (posts to GHL API)
GoHighLevel
    ↓ (schedules/distributes)
Social Media Platforms (Facebook, LinkedIn, Twitter, Instagram)
```

## Database Setup

### GHL Config Table

The `ghl_social_config` table stores GHL API credentials per site/profile:

```sql
CREATE TABLE ghl_social_config (
  id UUID PRIMARY KEY,
  site_id VARCHAR(50) REFERENCES sites(id),
  profile_name VARCHAR(100) NOT NULL, -- e.g., 'Keenan Shaw', 'Keenan Shaw'
  ghl_api_key TEXT NOT NULL, -- GHL JWT token
  ghl_location_id VARCHAR(100) NOT NULL,
  enabled BOOLEAN DEFAULT true,
  platforms JSONB DEFAULT '[]', -- ['facebook', 'linkedin', 'twitter', 'instagram']
  brand_voice TEXT,
  content_themes JSONB DEFAULT '[]',
  default_schedule_hours INTEGER DEFAULT 1,
  auto_post BOOLEAN DEFAULT false,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  updated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);
```

## Edge Functions

### 1. `ghl-social-poster`

**Purpose**: Posts article content to social media via GHL API

**Request**:
```json
{
  "article_id": "uuid",
  "site_id": "smallbizsimple",
  "profile_name": "Keenan Shaw", // optional if only one profile per site
  "platforms": ["facebook", "linkedin"], // optional, uses config default
  "schedule_hours": 1, // optional, uses config default
  "include_image": true, // optional, default: true
  "custom_content": "override content" // optional
}
```

**Response**:
```json
{
  "success": true,
  "posts_scheduled": 2,
  "platform_results": [
    {
      "platform": "facebook",
      "success": true,
      "post_id": "ghl_post_id",
      "scheduled_at": "2025-11-14T12:00:00Z"
    },
    {
      "platform": "linkedin",
      "success": true,
      "post_id": "ghl_post_id_2",
      "scheduled_at": "2025-11-14T12:00:00Z"
    }
  ]
}
```

## Troubleshooting

### "No GHL config found"

- Verify config exists in `ghl_social_config` table
- Check `site_id` matches article's `site_id`
- Ensure `enabled = true`

### "Platform not connected in GHL"

- Verify platform account is connected in GHL dashboard
- Check OAuth permissions are granted
- Re-authorize connection if needed

### "GHL API error: 401"

- Verify JWT token is valid and not expired
- Check token has proper permissions
- Ensure location ID matches token

### Posts scheduled but not appearing

- Check GHL dashboard for scheduled posts
- Verify platform accounts are active
- Check GHL API logs for errors

## API Reference

### GHL API Endpoints Used

- `GET /social-media-posting/{location_id}/accounts` - Get connected accounts
- `POST /medias/upload-file` - Upload image to media library
- `POST /social-media-posting/{location_id}/posts` - Schedule post

```json
{

  "type": "post",
  "userId": "oauth_id",
  "scheduleDate": "2025-11-14T12:00:00.000Z"
}
```
