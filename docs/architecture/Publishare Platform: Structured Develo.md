# Publishare Platform: Structured Development Instructions

## Core Platform Vision
Transform the user experience of an agentic content platform that guides users through strategic content creation workflows aligned with the customer value journey stages.

## 1. User Onboarding & Profile Setup (Reverse Order Priority)

### Step 1: Business Profile Configuration
- **Business Type Selection**: Present dropdown/cards for supported industries
  - Financial Advisor
  - Life Insurance 
  - Annuity
  - Mortgage
  - Real Estate
  - Fiduciary
  - Other (expandable)

### Step 2: Target Audience Definition
- **Persona Selection**: Dynamic avatar system based on business type
  - Seniors
  - First-time buyers
  - Young professionals
  - High-net-worth individuals
  - Small business owners
- **Persona Customization**: Allow fine-tuning of messaging tone and approach

### Step 3: Knowledge Base Integration
- **Business Information Capture**: 
  - Company details, specializations, unique value propositions
  - Compliance requirements and restrictions
  - Brand voice and messaging guidelines
  - Existing content library import

### Step 4: Content Strategy Goals
- **Journey Stage Selection**: Clear visual pathway showing:
  - **Awareness → Engagement** (Blog content + social amplification)
  - **Engagement → Subscription** (Interactive lead magnets)
  - **Subscription → Conversion** (Nurture newsletters)

## 2. Content Creation Workflows

### Awareness → Engagement Track
**Content Options:**
1. **New Article Creation**: Manual content input with AI enhancement
2. **AI-Generated Article**: Prompt-based content generation
3. **Generic Workflow**: Template-based approach (Pro users)

**Output:** Long-form content + automatic social promotion asset generation

### Engagement → Subscription Track
**Interactive Asset Creation:**
1. **Quiz Builder**: 
   - Type selection interface
   - Question suggestion engine
   - Custom results pages
   - Email capture integration
2. **Calculator Builder**:
   - Industry-specific templates
   - Custom calculation logic
   - Lead capture forms
   - Embed/shortcode generation

### Subscription → Conversion Track
**Newsletter Creation:**
1. **Content Curation**: Select from previously created blog posts
2. **Dynamic Newsletter Generation**: 
   - Digest format
   - Periodical style
   - Custom newsletter import option
3. **Automated Scheduling**: Delivery cadence settings

## 3. Enhanced User Interface Architecture

### Primary Navigation Structure
```
┌─ Quick Start Dashboard (Primary Landing)
├─ Content Creation Hub
├─ Analytics & Performance
└─ Settings & Profile
```

### Quick Start Dashboard Features
- **Workflow Launcher**: Visual journey stage selector
- **Recent Activity**: Last 5 actions with quick resume
- **Performance Snapshot**: Key metrics at-a-glance
- **Next Suggested Actions**: AI-powered workflow recommendations
- **Template Library**: Quick access to popular formats

### Content Creation Hub (CMS)
- **Active Projects**: Work-in-progress content
- **Content Library**: Organized by type and journey stage
- **Asset Manager**: Images, videos, brand materials
- **Workflow Templates**: Reusable content structures
- **Collaboration Tools**: Team access and approval workflows

### Analytics Dashboard
- **Content Performance**: Engagement metrics by journey stage
- **Lead Generation**: Conversion tracking for interactive assets
- **Audience Insights**: Persona engagement patterns
- **ROI Analysis**: Content effectiveness scoring

- **Seasonal Prompts**: Industry-specific timely content suggestions
- **Competitive Intelligence**: Content opportunity identification

### Content Strategy Visualization
- **Journey Mapping**: Visual representation of customer path
- **Content Calendar**: Strategic publishing timeline

### Enhanced Asset Creation
- **Template Marketplace**: Industry-specific starting points
- **Brand Consistency Engine**: Automatic style application
