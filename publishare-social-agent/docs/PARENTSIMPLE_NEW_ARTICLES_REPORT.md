## How to Find These Articles in the Database

### Query 1: Get All Early Years and Middle School Articles

```sql
SELECT 
  a.id,
  a.title,
  a.slug,
  a.status,
  a.created_at,
  ux.slug as ux_category_slug,
  ux.name as ux_category_name,
  a.featured_image_url,
  a.html_body,
  a.canonical_url
FROM articles a
INNER JOIN article_ux_categories auc ON a.id = auc.article_id AND auc.is_primary = true
INNER JOIN ux_categories ux ON auc.ux_category_id = ux.id
WHERE a.site_id = 'parentsimple'
  AND ux.slug IN ('early-years', 'middle-school')
  AND a.created_at >= '2025-12-01'
ORDER BY a.created_at DESC;
```

## Database Schema Reference

### Key Tables

1. **`articles`** - Main article table
   - `id` - UUID
   - `title` - Article title
   - `slug` - URL slug
   - `status` - 'draft' | 'published'
   - `html_body` - Rendered HTML (use this for display)
   - `content` - Markdown source (don't use for display)
   - `featured_image_url` - Image URL
   - `canonical_url` - Canonical URL

2. **`article_ux_categories`** - Junction table
   - `article_id` - Links to articles.id
   - `ux_category_id` - Links to ux_categories.id
   - `is_primary` - Boolean (true for primary category)

3. **`ux_categories`** - UX category definitions
   - `id` - UUID
   - `slug` - 'early-years', 'middle-school', etc.
   - `name` - Display name
   - `site_id` - 'parentsimple'

4. **`articles_with_primary_ux_category`** - View (recommended)
   - Combines articles + UX categories
   - Use this for queries instead of manual joins
